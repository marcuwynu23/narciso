## Bus Factor

Currently the bus factor is **1**. This means if the sole maintainer becomes unavailable, the project may stall. We are actively looking for additional maintainers to reduce this risk.

## Becoming a Maintainer

We welcome new maintainers! If you're interested:

1. Start by contributing — open PRs, review code, help with issues
2. Demonstrate reliability over several contributions
3. Express interest to the current maintainer via email or GitHub discussion

There's no strict timeline. Contributors who show consistent, high-quality work are invited to become maintainers.

## Maintainer Responsibilities

- Review and merge pull requests
- Triage and respond to issues
- Cut releases and tag versions
- Enforce the code of conduct
- Keep dependencies up to date
- Guide the project roadmap
