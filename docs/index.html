Narciso

A lightweight PHP web library for routing, middleware, database, and APIs. Build simple backends without a full framework.

Documentation View on GitHub

Install

composer require marcuwynu23/narciso

Clean URLs & pages

Map friendly addresses to your logic. Use dynamic segments (e.g. a user or post ID in the URL) and respond with JSON or HTML.

Security & control

Add security headers, control who can call your app from other sites, and limit how often requests can be made to protect your server.

Database ready

Connect to MySQL or SQLite and start reading and writing data. No extra setup—just point to your database and go.

APIs & HTML

Return JSON or XML for APIs, render HTML from PHP templates, and send users to the right page with redirects.
